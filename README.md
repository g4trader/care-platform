# Care Platform – MVP

Plataforma web para cuidadores (idosos, crianças, necessidades especiais) com certificação, marketplace e gestão de serviços.

## Estrutura

- `frontend/`: Next.js (deploy na Vercel)
- `backend/`: Node.js/Express + Prisma (deploy no Cloud Run)
- `docs/`: documentação de produto

## Comandos

```bash
npm run dev:frontend
npm run dev:backend
```
